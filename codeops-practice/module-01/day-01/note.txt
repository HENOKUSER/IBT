$ git --version   returns a version number,
$ git config --global user.name "Your Full Name"  Sets the name attached to every   
                                                 commit you make on this machine
$ git config --global user.email "you@example.com"  Sets the email attached to 
                                                    every commit 
$ git config --list      Displays all currently active configuration settings       
git add <file>                   Working Directory → Staging Area
git commit                    Staging Area → Local Repository
git restore --staged <file>      Staging Area → Working Directory (unstage)
git restore <file>       Local Repository → Working Directory (discard local edits)
git push                     Local Repository → Remote Repository
git pull / git fetch        Remote Repository → Local Repository 
git clone URL
